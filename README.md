#iFeedBird
iFeedBird est un projet basé sur Raspberry Pi qui permet de prendre en photo les oiseaux s'approchant d'une mangeoire. :bird:

###Matériel :
* Raspberry Pi A+
* PiCamera
* Capteur infrarouge

###Back :
* Flask
* SQLAlchemy

###Front :
* Backbone.Marionette
* HTML5/CSS3

###Database :
* SQLite
